歡迎來到北藝新媒！

國立臺北藝術大學「新媒體藝術學系」為全國首創，是一跨越人文藝術領域與尖端科技結合的未來學門，它作為推動數位科技與藝術整合的重要推手，以培育前瞻性的「新媒體科技」、「新媒體影像」與「新媒體跨域」等全方位未來藝術創意人才。 國立臺北藝術大學於1992年正式獲教育部核准成立全臺第一個專注於科技藝術領域的「科技藝術研究中心」。為延伸本校科技藝術研究中心歷年的前瞻性發展基礎，並基於科技與藝術整合的人才培育使命，2000年向教育部申請成立「科技藝術研究所」，並於2001年開始招收第一屆學生，主要目標為培養臺灣科技與藝術的跨領域創作與技術研發專業人才。經歷多年的努力，畢業學生的專業成就已具體展現並引領臺灣科技藝術各個領域的連動發展。在整合性延伸教育機制的思考下，於2009年奉准成立大學部，其後並與電影創作學系及動畫學系共同成立「電影與新媒體學院」，原「科技藝術研究所」同時更名為「新媒體藝術學系」，下設「學士班」與「碩士班」，於2010年正式招收第一屆大學部學生。
